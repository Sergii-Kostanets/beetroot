Дополнительноe задание со звёздочкой:

Создать Makefile в котором добавить следующие команды up для создание контейнров и поднятия проекта, down - для остановки и удаления контейнеров, start для запуска контейнеров, stop для остановки. При запуске make без параметров по молчанию должна выполняться команда up.

На следующий урок установите приложение Gnu Make и добавьте путь к установке и папке bin в переменную окружения в windows под названием PATH. (Для пользователей linux и mac ничего делать скорее всего не нужно).
